# ParamorphismDecrypt

Usage JDK11+

-Decrypt the encrypted class created by Paramorphism.  
-Show the class hidden as folders.  
-Show the class with "empty" name.  

-解密由Paramorphism混淆器生成的存储字符串及调用的类  
-显示以文件夹形式隐藏的类  
-显示在解压缩软件中被展示为空名字的文件
